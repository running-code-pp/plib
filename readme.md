# 简介
plib 是一个跨windows/linux 的c++工具库，设计目标是为了一个容易集成和使用，简洁高效的c++工具库，设计目标是为了一个容易集成和使用，简洁高效的c
可选择式引入，甚至于拿走某些文件就能直接用，所以plib中各个部分的耦合度是非常低的，不同于folly

plib提供了如下功能：
- 并发操作相关
- 配置相关
- 文件操作相关
- 日志
- 内存
- 反射
- 崩溃报告
- 网络
- 插件系统
- 设计模式抽象
- 常用工具函数
- frame
- coroutine
